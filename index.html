Projects

TechnoInvaders


TechnoInvaders is my best project at the moment and the one I'm most proud of. Its an arcade game for Android inspired by Space Invaders but modernised and with some rogue-lite elements sprinkled in. There is still a ways to go with this project and I plan to do a complete overhaul of the graphics, but its getting there and I'm hoping to release it at some point when I'm not busy with uni. Click the button to the right to go to the Github for this project. There you can find all the files for the game and a .pak file for Android.

Shader Test


This shader shown in the video is from testing and figuring out the shader graph tool in Unity which allows you to visually script shaders. For this project I've created an ocean shader (Shown to the left), and an outline pulsating glow shader. I plan to create more shaders in this project that I'll be able to use in other projects in the future.

FPS Test


At the moment this is a simple FPS where you shoot the enemies until they all die. The enemies have some "AI" but its very basic and predictable at the moment. I have started implementing multiplayer and so far I've been able to get LAN working with UNet. When I get more time I will probably continue working on this as it was quite a fun project. Click the icon to the right to go the the Github for this project.

Crates


Crates is a 2d platformer where you have to get to the top of the container without getting hit by the falling crates. At the moment there are two levels and it has a main menu that allows you to switch between them. I may continue developing this one and eventually release it, most likely for Android. Click the icon to the right to go the the Github for this project.

Frogger3D


Frogger3D is similar to frogger where you have to cross a road, but when its complete it will be an infinite runner set in 3d. You have to collect all the coins in each section to advance to the next section and with each section the cars will get faster. So far I've gotten two sections and im working on making it infinite. Click the icon to the right to go the the Github for this project.

Endless Roller


This is a game I made at University. Its an endless puzzle-platformer where you have to get a ball that you don't control as far as possible by shooting boxes as objects in the level. At the moment the basics are in and working. I just need to add more puzzles to the game which I've made extremely easy to integrate into the random generator. I'm still working on this game and might release it fully, probably on Android as mobile is a much better platform for endless games like this. Click the icon to the right to go the the Github for this project.
